AWS Static Website Deployment

Hosted on Amazon S3 | Delivered with CloudFront

Project Overview

This project demonstrates how to deploy a static website using Amazon S3 and distribute it globally using CloudFront. It showcases cloud storage, CDN integration, and secure public access configuration.

Key Features

  • Static Website Hosting on S3
  • CloudFront CDN Distribution
  • Bucket Policy Configuration
  • Global Content Delivery
